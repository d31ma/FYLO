Fylo Explorer (FX)

A local, in-browser workbench for your FYLO document store. Point it at a FYLO root on your machine and browse, query, and edit it — everything runs client-side through the File System Access API. Nothing is uploaded, and reads never modify your folder. Available in browsers that support the File System Access API, including Chrome, Edge, and Brave. To explore a remote FYLO root, first mount it locally (for example via FTP, SFTP, or SMB), then select that mounted folder.

  • Browse collections & buckets — documents, raw files, and folder trees
  • Query with SQL and PostgREST filters, nested keys included, accelerated in a browser worker with Wasm and automatic JavaScript fallback
  • Edit documents and files, with autocomplete and inline diagnostics
  • Inspect git-like version history and a write-activity overview
